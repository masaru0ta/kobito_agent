# 機能一覧

| # | カテゴリ | 機能名 | 概要 | 仕様書 |
|-----|---------|--------|-----|--------|
| 1-1 | 基盤 | config | エージェント定義（config.yaml, CLAUDE.md）の読み込み・バリデーション | `spec_config.md` |
| 1-2 | 基盤 | runner | LLM呼び出しエンジン。プロンプト組み立て→LLM実行→応答返却 | `spec_runner.md` |
| 1-3 | 基盤 | chat | ユーザー×エージェントのチャット処理。会話履歴の保存・取得 | `spec_chat.md` |
| 1-4 | 基盤 | web | Web UI。エージェント選択、チャット | `spec_web.md` |
| 2-1 | 設定管理 | settings | Web UIからconfig.yaml・CLAUDE.md・mission.md・task.mdを編集 | 未着手 |
| 3-1 | ミッション・タスク | runner（自律） | mission/taskを読み、作業を実行し、成果物を出力する | 未着手 |
| 4-1 | トリガー | trigger（cron） | 定期トリガーで自律思考サイクルを回す | 未着手 |
| 5-1 | 通信 | trigger（エージェント間） | エージェント同士がトリガーで会話する | 未着手 |
| 6-1 | 記憶 | memory | Mem0による記憶の保存・想起 | 未着手 |
