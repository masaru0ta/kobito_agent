# 記憶システム仕様書の作成

**作成日**: 2026-03-25
**対象**: フェーズ6 - Mem0記憶システム
**成果物**: `docs/spec_memory.md`

## 背景

kobito_agentのコンポーネント一覧で唯一未着手だった記憶システム（memory）の仕様書を作成。CLAUDE.mdの設計思想と既存仕様書のフォーマットに基づいて、Mem0を使用した記憶システムの詳細仕様を策定した。

## 作成した仕様書の概要

### 主要機能
1. **記憶の保存**: エージェントの思考・会話から自動記録
2. **記憶の検索**: ベクトル類似検索による関連記憶の取得
3. **自動想起**: 思考サイクル・チャット時の文脈関連記憶の自動想起
4. **エージェント分離**: 各エージェント専用のメモリストレージ

### 技術仕様
- **ベクトルストア**: Chroma（デフォルト）
- **埋め込みモデル**: text-embedding-ada-002
- **API**: REST APIでWeb UI連携
- **パフォーマンス**: 検索500ms以内、保存100ms以内

### 設計思想の反映
- **エージェント自律性**: 何を記憶するかはエージェント判断に委ねる
- **プライバシー分離**: エージェント間で記憶を共有しない
- **シンプルな仕組み**: 複雑なルールを避け、保存・検索機能に特化

## 開発への影響
